# Sistema de Gestión de Stock - ISFDyT N° 210

## Descripción del Proyecto
Este sistema es un desarrollo web realizado individualmente para el **Trabajo Práctico Integrador de Consolidación de Temas (2026)** de la carrera **Tecnicatura Superior en Análisis de Sistemas** (3° Año).

El objetivo principal es resolver la problemática de control de inventario, permitiendo administrar productos, categorías y registrar movimientos de stock de entrada y salida, incluyendo validaciones de consistencia[cite: 8].

## Tecnologías Utilizadas
* **Backend:** Python 3 & Django Framework[cite: 8]
* **Frontend:** HTML5, CSS3 (Bootstrap 5 & Bootstrap Icons)[cite: 8]
* **Base de Datos:** SQLite3[cite: 8]

## Funcionalidades Principales
* **Dashboard Principal:** Panel de control con métricas clave del estado actual del stock[cite: 8].
* **Gestión de Productos:** Alta, baja, modificación y listado de artículos[cite: 8].
* **Gestión de Categorías:** Clasificación y organización de productos[cite: 8].
* **Registro de Movimientos:** Control estricto de entradas y salidas de stock con validación ante faltantes[cite: 8].
